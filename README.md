# 좌충우돌 카페 설립기 — 코드 저장소

> AI 에이전트 발전사를 카페 창업 1인칭 체험기로 풀어낸 블로그 시리즈의 실습 코드 모음.
> 매 화마다 카페 운영 문제를 AI로 해결하는 코드가 쌓이고, 시즌 마지막에 하나의 에이전트(헤르메스)가 완성됩니다.

## 시작하기

```bash
git clone https://github.com/worldlotus/ai-cafe-series.git
cd ai-cafe-series
pip install -r requirements.txt
```

## 마스터 코드베이스

`master/` 폴더에는 각 화의 재사용 가능한 함수·모듈이 누적됩니다.
시즌 마지막에는 이 폴더가 하나의 작동하는 에이전트가 됩니다.

```bash
python -c "import master; print('헤르메스 에이전트 준비 완료')"
```
